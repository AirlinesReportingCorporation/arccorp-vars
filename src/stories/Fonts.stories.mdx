# Classes
## How to use Font and Color classes

These are meant to make it easier to add any weight of font, color and background color.

## Typography
<div className="type-font-light"> Light</div>
<div className="type-font-regular"> Regular</div>
<div className="type-font-semibold"> Semibold</div>
<div className="type-font-bold"> Bold</div>
<div className="type-font-black"> Black</div>

To use the className: type-font- (insesrt thickness here) <br/>
By default, the font is SourceSansPro. 
```
<div className="type-font-light">Light</div>
<div className="type-font-regular">Regular</div>
<div className="type-font-semibold">Semibold</div>
<div className="type-font-bold">Bold</div>
<div className="type-font-black">Black</div>
```
