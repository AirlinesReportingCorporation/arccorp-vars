## Font Colors

### Getting started

To use text colors, you can import colors from arccorp-vars by using the use statement:

```
@use "~arccorp-vars/src/scss/colors.scss";
```

To use all parts of the arccorp-vars system, you can import main:

```
@use "~arccorp-vars/src/scss/main.scss";
```

arccorp-vars uses a list of predefined colors that are aligned with our branding. You can see more in our style guide listed <a href="https://www2.arccorp.com/styleguide">here</a>.

### Using Font Colors in sass

Additionally, you can use these colors inside your sass pages by doing the following:

```
color: main.$black;
```

This uses the scss variable that aligns with the classes listed above. Make sure that the arccorp-vars is imported using the `@use` statement shown at the top of this page.

You can also use these to make border colors, as you would with any other color:

```
border-bottom: 1px solid main.$tarmac;
```
